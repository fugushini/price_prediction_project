# Классификация логов с использованием гибридной системы классификации

Этот проект реализует предсказание цены на недвижимость


## Структура папок

1. **`model/`**:
   - Содержит код для обучения моделей с использованием XGBRegressor.
   - Включает описание данных.

2. **`client/`**:
   - Хранит хранит frontend проекта js+html+css.

3. **`server/`**:
   - Хранит backend проекта flask и вспомогательные методы в util.


## Использование
- Собрать докер файл: docker build -t server-service:latest
- Запустить: docker run -d --name server-service -p 500:5000 server-service:latest
